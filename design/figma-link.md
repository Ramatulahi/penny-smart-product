# Product Design
The design process for **Penny Smart** followed a structured product design approach starting from low-fidelity wireframes to an interactive prototype.

---
# Low Fidelity Wireframes
Initial wireframes were created to map out the core user flows and application structure.

Low-fi design link:
https://www.figma.com/design/Ijad8EaQpGA2m7f3HNLweD/Untitled?node-id=0-1&t=fLkqyb5jpu29hDCe-1

These wireframes helped define:
- Dashboard layout
- Savings goal flow
- Budget management structure
- Transaction tracking screens
