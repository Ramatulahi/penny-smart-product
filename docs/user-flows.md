# User Flows

## Onboarding Flow

1. User downloads or accesses the application
2. User creates an account
3. User logs into the dashboard
4. User sets up initial financial categories

---

## Transaction Flow

1. User adds income or expense
2. User selects category
3. Transaction is saved
4. Dashboard updates automatically

---

## Savings Flow

1. User creates a savings goal
2. User sets target amount and date
3. User adds funds periodically
4. Progress is tracked visually
